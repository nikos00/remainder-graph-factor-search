# Recovered experiment outputs

## `aggregate_10_digit`

These seven CSV files are the surviving aggregate outputs from the original 10-digit experiment set. In particular:

- `stats_fraction_events.csv` stores individual successful events and centers;
- `stats_run_summary.csv` stores per-run parameters and totals;
- the remaining CSVs provide fraction-level classifications and summaries.

Experiment ID 9 reproduces the paper's 10-digit row exactly.

## `raw_24_digit_later_rerun`

These are 22 raw `.out.txt`, `.meta.txt`, and `.progress.txt` files from a later 24-digit experiment using 9,900-second configurations. They were preserved because they are genuine recovered outputs, but they are not the original 1,000-second outputs used for the paper's 24-digit table row.

Original raw outputs for the remaining paper rows were not found in the copied `python.v2/flat_v2` directory.
